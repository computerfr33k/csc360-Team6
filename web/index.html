OnTrack is a task manager design with school type tasks in mind. OnTrack allows you to sort your tasks in the table below by clicking on the column header. If you want to sort with multiple columns, just hold shift while clicking on the other tab.
To add a task, just fill out the form on the top where it says Add Task and then click the add task button once your are done filling it out. The task will then be added to the table below.
To edit a task, start by selecting a row from the table with the task that you want to change. Then click on the Edit Selected Task button above the table. This will then put the existing task information into the form on top where you can make changes and then save them back.
To Delete a task, select the row with the task you want to remove and then click on the Delete Selected Task Button above the table. The task will then be removed.
